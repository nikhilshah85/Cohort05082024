				             HTML

				Hyper Text Markup language
			--UI content development language
			--understood by Browsers (IE, Chrome, Edge, Safari.....)
			--Governed by W3C (World Wide Web consortium)
			--text based language 
			--W3C has defined tags for various visual (img, paragraph, videos, forms etc)
			--developers writes this tags in < > -- triangle brackets
			--browsers has an engine - called Parser which will interpret tags and show visuals
		
			--to beautify HTML, we use CSS (cascading stylesheet)
			--to make HTML interactive with end users we use Javascript --> jquery
		--we create an .html document to display visual and inside the document we write tags
			--every document starts with <!DOCTYPE HTML>
			--html document is divided in various sections

			<html>
				<head>
					--information on how to load page
				</head>
					
				<title> Home Page </title>
				
				<body>
					--contents for visuals
				</body>			
			</html>
	
			--every tag will have an matching closing tag <h1> ... </h1>	
									<p> ... </p>
			--some tags does not have closeing tags,they are self closed like </br>
			 <!-- for responsiveness -->
